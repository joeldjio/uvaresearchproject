# RZ GCS 0.3.0

RZ GCS 0.3.0 release.

## Highlights

- Public update/release channel moved to `joeldjio/rz-gcs-releases`.
- Windows installer for RZ GCS.
- License contact: `djiojoel2@gmail.com`.
- Trial and offline license-key system included.
- In-app updater checks the public release repository.
- Swarm tab includes system information for selected drones.
- COM-port refresh/listing support documented.
- Project overview updated with release, license, update, and repository workflow notes.

## Installer asset

Upload this file as release asset:

```text
RZ-GCS-Setup-0.3.0.exe
```

The updater only detects assets whose name starts with `RZ-GCS-Setup-` and ends with `.exe`.
